##### 目录说明：

- 家族树
  - bootstrap:bootstrap框架文件
  - css:网页的css样式文件
  - images:插入的图片
  - js:html文件中需要导入的一些js文件
  - type：html中需导入的css文件
  - 学术家族树：html文件  
  - README:目录说明与使用说明文件

##### 使用说明：

- 先点击github上的“Clone or download”按钮，下载完后解压并用浏览器（建议使用谷歌浏览器）打开 **新建家族树.html**，输入作业要求文字即可。
- 在右下方的文本框输入相应数据，点击生成家族树按钮，将会在左边生成一棵以导师为根节点的树（暂不支持关联树的实现)，然后点击刷新按钮，可以刷新页面重新输入。
